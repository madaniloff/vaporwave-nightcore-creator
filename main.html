How to Use The Vaporwave and Nightcore Creator:

First of all, what are Vaporwave and Nightcore?

To use this webpage, first find an audio file that you would like to modify. Click the 'Choose File' button and select the file. To the right of the upload button is an audio player, which allows you to play the file that you uploaded.

Below, there are two sliders, one for speed, and one for pitch. Moving the first slider left slows the audio down, while moving it right speeds audio up. For the second slider, moving it lefts decreases the pitch, while moving it right increases the pitch.
